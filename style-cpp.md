# C++ Coding Style

[ES.11](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#res-auto):
Use auto to avoid redundant repetition of type names

[ES.20](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#res-always):
Always initialize an object

[ES.46](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#res-narrowing):
Avoid lossy (narrowing, truncating) arithmetic conversions

[ES.49](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#res-casts-named):
If you must use a cast, use a named cast

[I.2](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#ri-global):
Avoid non-const global variables

[SL.con.1](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#rsl-arrays):
Prefer using STL array or vector instead of a C array
